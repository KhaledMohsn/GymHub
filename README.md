
# GymHub — Gym Management Platform

A full-stack, multi-tenant platform built to modernize gym operations. GymHub replaces paper records and WhatsApp group chaos with a single professional system for member management, subscription tracking, and attendance.

---

---

## Features

- **Multi-tenant architecture** — one platform, unlimited gyms, zero data leakage between tenants
- **Member management** — add, update, activate/deactivate members with confirmation guards
- **Subscription plans** — create custom plans per gym, assign to members with automatic expiry calculation
- **Attendance tracking** — one-click check-in with automatic subscription validation
- **Staff management** — owner, trainer, and receptionist roles with enforced permissions
- **JWT authentication** — secure token-based auth with role-based access control
- **Centralized logging** — all events and errors logged to file and console
- **Global error handling** — clean JSON error responses, no raw stack traces exposed
- **Fully containerized** — Docker + Docker Compose for one-command deployment
